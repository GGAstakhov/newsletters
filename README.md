# **Сервис управления рассылками**

- Интерфейс системы содержит следующие экраны: 
- список рассылок, 
- отчет проведенных рассылок отдельно, 
- создание рассылки, 
- удаление рассылки, 
- создание пользователя, 
- удаление пользователя, 
- редактирование пользователя.
